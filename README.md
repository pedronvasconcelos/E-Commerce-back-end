
# A ecommerce application with multiple APIs built with ASP.NET 6


---


## Technologies / Components implemented

- .NET 6
    - ASP.NET MVC Core
    - ASP.NET WebApi
    - ASP.NET Identity Core
    - Refresh Token
    - JWT    
    - Background Services
    - Entity Framework Core 6

- Components / Services
    - RabbitMQ
    - EasyNetQ
    - Refit 
    - Polly
    - FluentValidator
    - MediatR
    - Swagger UI with JWT support
